# 🍕 App Restaurante

Aplicativo Android para gerenciamento de mesas.

## 📱 Funcionalidades
- Seleção de mesas
- Reservar mesas
- Remover reservas
- Salvamento com SharedPreferences

## 🛠 Tecnologias
- Java
- Android Studio
